丰华生物体系（丰华生物 + 天力通 + 悦通 + 加拿大） 丰华制药（江苏制药） 同名不同岗的部门

三级关系：

管理体系 (Company.managementGroup) → 部门 (Department) → 岗位 (Position)

核心逻辑：

  • 每个部门属于唯一管理体系
  • 每个岗位通过 departmentId 直连一个部门
  • 同名的行政人事部在两个体系下岗位完全不同
  • EmployeeDepartmentPosition 是员工挂在这个三级树末梢的记录
