## 已验证的计算逻辑

### 1. 成本波动率分析

**公式：**
```
波动率下降 = (2024年成本波动率 - 2025年成本波动率) / 2024年成本波动率
```

**验证结果：** ✓ 完全匹配（所有样本验证通过）

**示例：**
- 盐酸左氧氟沙星胶囊 0.1g*9粒*2板：
  - 2024=0.2492, 2025=0.1119
  - 计算：(0.2492 - 0.1119) / 0.2492 = 0.5508
  - Excel：0.5508 ✓

---

### 2. 单位成本比较表

**公式：**
```
单位成本节约额 = 2024年单位成本 - 2025年单位成本
单位成本节约百分比 = 节约额 / 2024年单位成本
```

**验证结果：** ✓ 完全匹配（所有样本验证通过）

**示例：**
- 别嘌醇28片：
  - 2024=4.3990, 2025=3.1516
  - 节约额 = 4.3990 - 3.1516 = 1.2475 ✓
  - 百分比 = 1.2475 / 4.3990 = 0.2836 ✓

---

### 3. 毛利率表

**列结构：**
| 列 | 说明 |
|---|---|
| 单价 | 含税销售单价 |
| 不含税单价 | 单价 / 1.13 |
| 成本 | 含税单位成本 |
| 不含税成本 | 成本 / 1.13 |
| 毛利 | = 不含税单价 - 不含税成本 |
| 毛利率 | = 毛利 / 不含税单价 |

**验证结果：** ⚠️ 部分匹配

- **阿奇霉素胶囊 0.25g*8粒：** ✓ 完全匹配
  - 不含税单价=3.1858, 不含税成本=3.0939
  - 毛利 = 3.1858 - 3.0939 = 0.0920 ✓
  - 毛利率 = 0.0920 / 3.1858 = 0.0289 ✓

- **阿替洛尔片 25mg*60片*480瓶/件：** ✗ 不匹配
  - 不含税单价=2.3439, 不含税成本=1.8157
  - 计算毛利 = 0.5281，但 Excel 显示 1.4
  - 计算毛利率 = 0.2253，但 Excel 显示 0.5973

**待确认问题：**
> 阿替洛尔片、别嘌醇片等产品的"毛利"列不是简单的"不含税单价 - 不含税成本"。
> 可能原因：
> 1. "成本"列来源不是成本结构表（可能是其他口径）
> 2. 毛利是手动填入的固定值
> 3. 存在某种单位换算（如按件/按瓶计算差异）
